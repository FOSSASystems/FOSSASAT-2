105/*
106 This CRC can be specified as:
107
108 Width : 32
109 Poly : 0x04c11db7
110 Init : parameter, typically 0xffffffff
111 RefIn : false
112 RefOut : false
113 XorOut : 0
114
115 This differs from the "standard" CRC-32 algorithm in that the values
116 are not reflected, and there is no final XOR value. These differences
117 make it easy to compose the values of multiple blocks.
118 */
